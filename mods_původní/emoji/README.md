# emoji

"Emoji" show emoji icon above player with command "/e"
or with text smiles in chat like 'o_O'

Available chat smiles to spawn emoji:
:) B-) :# *_* :D :* :_( >:-[ ]:-) :/ ;) :( ;P :'-D ~:[ o_O xD xP :X :P :O
